# Porsche Experience Website

A premium, immersive web experience showcasing Porsche's engineering and design.

## Features
- **Dynamic Engine Architecture**: Interactive 3D/2D engine visualizations.
- **Glassmorphic UI**: Modern, high-end aesthetic with frosted glass elements.
- **Vertical Scroll Services**: Immersive full-screen service exploration.
- **Experience Overlay**: Detailed deep-dives into Porsche Centers and Personal Commissioning.

## Tech Stack
- Next.js
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```
